Tags e seus atributos

ir para índice

As tags são parte fundamentas para o HTML, sendo que sem elas não podemos realizar nada no HTML, mas a dúvida principal é, o que são as tags? As tags são instruções as quais passamos ao navegador, demostrando a estrutura do texto que será utilizada, e assim podemos definir: títulos, parágrafos, imagem, links e etc..

Quando falamos sobre os atributos, podemos dizer que eles são meio que são características especias que aplicamos a uma tag especifica, por exemplo o atributo "href" com ele junto com a tag "a" podemos direcionar o usuário para links específicos, ou para uma parte do site, que queremos que ele vá automaticamente ao clicar, podendo ainda ser definido se a página vai ser carregada em um nova guia ou não, com outro atributo o "target"

Textos XHTML

ir para Índice

No HTML é importante termos um texto bem estruturado, para isso temos as tags de texto, vou fazer uma lista abaixo de algumas:

Elementos de texto HTML

  • h1 (O primeiro titulo deo seu texto
  • h2 (Seria subtitulo)
  • h3..h6(O subtitulo do subtitulo, ainda temos mais, as tags de titulo vão até h6)
  • p (tag destinada para marcar paragrafo)
  • blockquote(tag destinada para citações)

Com essas tags já podemos ter um texto bem estruturado, podemos dizer que já podemos fazer nosso primeiro texto HTML somente com esses tributos.

Algo que é bem comum em nosso cotidiano é fazer listas, e sim é possível realizar lista no HTML, mas ai temos uma peculiaridade. No caso temos dois tipos de listas, ordenadas e não ordenadas. A lista ordenadas são lista que temos um certo tipo de numeração, exemplo:

  1. Feijão
  2. Arroz
  3. Macarrão

Ainda assim também e possível fazer uma lista dentro de um outra lista, e para que usaríamos isso, para criar subtópicos

Exemplo:

  1. HTML
    1. Básicos
    2. Tags
  2. CSS
  3. JAVA

Mas para funcionar ainda precisamos colocar um atributo nesse tag, o atributo 'href' a qual é possível colocar dentro das aspas, um link para outro site pro exemplo, ou direcionarmos para um documento html por exemplo.

Ainda temos mais um atributo importante, o atributo 'target'. Com ele temos algumas opções que são mais utilizadas o target "_blank" a qual direciona o usuário em um nova guia web e target "parent" a qual direciona o usuário para um outra pagina só que na mesma guia.

Id como complemento de ancoragem com o atributo href

Nesse tópico vamos ver um complemento para a tag de ancoragem (a), vamos utilizar o id e o href juntos para criar um forma de direcionar o usuário para certa parte de nossa página Como deve ser feito? É algo bem simples, no texto a qual o vai direcionar o usuário para outra parte do site deverá ser utilizado o atributo 'href¨#NomeDoID' (precisa conter o #), e na parte do texto em qual o usuário vai ser direcionado ultimamos a atributo id, só que agora não precisamos colocar o '#'
